# Optimus Landing Page

A premium, high-performance landing page representing my personal website. This project showcases modern web design principles with a focus on speed, aesthetics, and user experience.

## Overview

Optimus is designed to be a state-of-the-art landing page with smooth animations, responsive layouts, and a cohesive design system. It is built as my personal website to demonstrate technical proficiency and design standards.

## Features

- Modern UI/UX: Vibrant colors, dark mode support, and glassmorphism effects.
- Dynamic Animations: High-performance animations using Framer Motion and custom CSS (styled-jsx).
- Responsive Design: Fully optimized for desktop, tablet, and mobile devices.
- Performance: Built with Next.js for server-side rendering and fast load times.
- Premium Aesthetics: Curated typography and harmonious color palettes for a professional feel.

## Technical Details

- Framework: Next.js 16+
- Language: TypeScript
- Styling: Vanilla CSS and TailWind CSS
- Components: Shadcn UI based custom components
- Icons: Lucide React
